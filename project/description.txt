Проект содержит 4 файла в соответствии с заданием. Чтобы запустить меню необходимо использовать команду ./menu. В результате открется диалог с выбором файла. Для файлов remftrash и unftrash необходимо указат файлы для восстановления. Для отальных вариантов дополнительных парамметров вводить не ужно.


Особенности реализации некоторых программ:
1. При удалении файлов с помощью remftrash имя файла заменяется на "старое имя файла" + "_" + номер, с помощью которого имя файла будет уникальным. Благодаря этому никогда не возникает конфликов при выполнении программы, однако для того, чтобы восстановить файл с помощью unftrash необходимо указать новое имя файла. Имя восстановленного файла будет то же, что и в файле trash.log

2. При использовании команды restorebackup будут скопированы все файлы, поскольку имя файла до бекапа может содержать дату, в следствие чего я не нашел более удовлетворительного способа решить эту проблему
